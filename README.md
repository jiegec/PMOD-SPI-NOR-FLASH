# PMOD-SPI-NOR-FLASH

PMOD expansion board with SPI nor flash(W25Q128JVSIQTR).

PMOD pin definition:

1. CS# with pull-up resistor
2. IO0/D0/DI/MOSI
3. IO1/D1/DO/MISO
4. CLK with pull-down resistor
5. GND
6. VCC
7. NC
8. NC
9. IO2/D2/WP# with pull-up resistor
10. IO3/D3/HOLD# with pull-up resistor
11. GND
12. VCC
